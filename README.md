Stephen Calculator - Advanced Scientific Calculator

A powerful, sleek, and intuitive Android scientific calculator built with Java. This app features a modern dark-themed UI and a robust expression evaluation engine capable of handling complex mathematical operations.

Features
•Advanced Math Engine: Supports a wide range of scientific functions including:
◦Trigonometry: sin, cos, tan (Degrees).
◦Logarithms: log (base 10) and ln (natural log).
◦Power & Roots: Square root (√), exponents (^), and squaring (x²).
◦Calculus & Probability: Factorials (!) and percentage (%).
◦Constants: Direct support for π (Pi) and e.


•Live Preview: See the result of your expression in real-time as you type in the history area.


•Smart Expression Parser: Handles nested parentheses, standard order of operations (PEMDAS), and automatic closing of parentheses.


•Modern UI:
◦Sleek dark theme optimized for night use.
◦Large, high-contrast buttons for accessibility.
◦Color-coded keypad (Scientific, Operators, Digits, and Actions).


•History Display: Shows your last calculation alongside the current expression.

🛠 Tech Stack

•Language: Java
•UI Framework: XML Layouts with Material Components
•Minimum SDK: Android API 24+
•Build System: Gradle

Installation
1.Clone the repository:git clone https://github.com/StephenOwo/StephenCalculatorApp.git

2.Open in Android Studio:
◦Launch Android Studio.
◦Go to File > Open and select the cloned folder.

3.Build and Run:
◦Connect your Android device or start an emulator.
◦Click the Run button (green play icon) in the toolbar.

Usage
1.Basic Operations: Simply type your math expression (e.g., 10 + 5 × 2) and press =.
2.Scientific Mode: Use functions like sin(30) or log(100). The app automatically opens a parenthesis for you.
3.Delete/Clear:
◦Use ⌫ (Backspace) to delete the last character. It's smart enough to delete full function names (like sin() in one tap.
◦Use C to clear the entire screen.
4.History: The smaller text above the main display shows a live result of your current expression.
